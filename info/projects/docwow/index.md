# DocWow

A healthcare AI application built with my friend Cam in the summer of 2023. The core idea: build a system that could ingest scanned medical documents—the faxed PDFs that dominate healthcare back-office work—and enable back-office workers to chat with them and generate output from them with proper citations.

DocWow was my introduction to the healthcare RCM space, and it's not a coincidence that I ended up at Thoughtful AI shortly after finishing the prototype.
